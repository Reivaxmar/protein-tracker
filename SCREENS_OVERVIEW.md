### Example: Adding a Meal

1. User fills form in Add Meal screen
2. User taps "Add Meal" button
3. Screen validates input
4. Screen calls `addMeal()` from store
5. Store creates new meal with calculated protein
6. Store updates daily totals
7. Store saves to AsyncStorage
8. User navigates to Home screen
9. Home screen shows updated data
